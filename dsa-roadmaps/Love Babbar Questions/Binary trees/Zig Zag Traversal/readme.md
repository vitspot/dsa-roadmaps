Given a Binary Tree. Find the Zig-Zag Level Order Traversal of the Binary Tree.

 

Example 1:

        Input:
                3
            /   \
            2     1
        Output:
        3 1 2

Example 2:

        Input:
                7
                /     \
            9       7
            /  \     /   
            8    8   6     
        /  \
        10   9 
        Output:
        7 7 9 8 8 6 9 10
